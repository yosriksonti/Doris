== Description ==

This plugin asks the user to choose his role in the website during registration from a list of selected roles. It works with BuddyPress also. It provides an interface from where the admin can choose which roles to display during registration for the user to choose from.

== Installation ==

To install WP Member Roles at Registration

1. Upload `wp-roles-at-registration.php` to the `/wp-content/plugins/wp-roles-at-registration/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= How to choose roles to be displayed during registration? =

Go to wp-admin >> Settings >> WP Roles at Registration and choose the roles which will be displayed during registration.
