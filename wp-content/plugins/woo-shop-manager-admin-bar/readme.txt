Save time managing your WooCommerce shop! Shop Manager Admin adds a customizable WooCommerce quick-links menu to the WordPress admin bar (frontend & backend).

The plugin lets you customize the admin login page, hide admin dashboard widgets, change admin footer text, add WooCommerce columns to users admin page and more helpful options to make the WooCommerce admin easier to use.


== Features==

### General Options
  * Display shop manager menu in Admin toolbar
  * Admin Footer Text

### Dashboard Options
  * you can clear everything from the dashboard.
  * Show Dashboard Panels

### Login Page Options
  * You can customize to styling branded login page and login form to shop manager plugin setting.
  * Upload Logo
  * Logo Width
  * Logo Bottom Margin
  * Background Color
  * Font Color
  * Login Page Footer Text
  * Login Box Label Color
  * Login Box Button Color
  * Login Box Background Color

### Admin menu bar Options
  * You can customize to Shop menu tab setting which allows you to Show menus and Submenus.
  * Show Admin Bar Menu Items

### Woocommerce Options
  * You can customize to WooCommerce tab setting which help to add WooCommerce features in the admin.
  * Add Total Spend Column to users admin
  * Add Order Count Column to users admin
  * Add order counts for order status
  * Horizontal scroll in orders admin

== Installation ==

1. Upload the folder 'woocommerce-shop-manager-admin-bar` to the `/wp-content/plugins/` folder
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Make sure you have setup "Woocommerce Shop Manager Admin Bar Option" in general settings.
